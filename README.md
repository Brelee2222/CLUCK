1. Install all node dependencies
2. Change port/public ip in src/consts.ts if needed
3. Run `npm run build` and `npm run serve`!

Pages:
- `/grid/`
- `/dash/`

API:
- `POST /api/clock`
- `POST /api/log`
- `GET /api/loggedin`
- `GET /api/ping`
- `GET /api/members`
- `GET /api/members/refresh`
